# Formativa_2_BCY0010_002D

Para uso del encriptador:

Con un archivo de texto en la misma carpeta, ejecutar el comando "python crypto_tool.py cifrar <archivo_a_cifrar.txt>" en terminal

Tras la ejecución, se crea un archivo "secret.key" con la llave, y un archivo ".enc" con el mensaje del archivo de texto encriptado.

Para descifrar, se ejecuta el comando "python crypto_tool.py descifrar <archivo_a_descifrar.enc>" en terminal

Tras la ejecución, se crea un archivo ".dec.txt" con el mensaje descifrado.
